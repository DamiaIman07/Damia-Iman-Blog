# CSD 34203 Special Topics in Software Development
## Individual Project: GitHub Portfolio (Personal Blog Page)

### 1. Project Title
**Damia Iman Personal Blog & Portfolio Website**

### 2. Description
Laman web ini merupakan sebuah portfolio digital dan blog peribadi yang direka untuk memenuhi syarat tugasan individu bagi kursus CSD 34203 di Fakulti Informatika dan Komputer (FIK), UniSZA. Ia memaparkan pengenalan diri serta artikel perkongsian akademik berkaitan pembangunan bidang kreatif komputer.

### 3. Features
* **Premium Geometric Design:** Rekaan moden menggunakan gabungan warna profesional serta corak grid bertitik (*background pattern*) yang estetik.
* **Persistent Dark Mode Functionality:** Pengguna boleh menukar mod pencahayaan pelayar dan tetapan tersebut akan disimpan secara kekal merentasi semua halaman menggunakan `localStorage`.
* **Clean & Responsive Layout:** Antara muka estetik yang dioptimumkan sepenuhnya untuk paparan telefon pintar.

### 4. Technologies Used
* **HTML5** - Untuk rangka kerja dan struktur kandungan.
* **CSS3** - Untuk penggayaan visual, corak geometri, kesan transisi lembut, dan reka bentuk responsif.
* **JavaScript (ES6)** - Untuk logik interaktif storan sistem Dark Mode.
