# Damia Iman Portfolio & Personal Blog

## Description

This project is a personal portfolio and blog website developed for the CSD 34203 Special Topics in Software Development course. The website showcases my background as a Media Informatics student, highlights my technical skills, and presents projects related to web development, augmented reality (AR), virtual reality (VR), and multimedia applications.

## Features

* Responsive design for desktop and mobile devices
* Modern dark mode interface
* Smooth scrolling navigation
* Personal profile section
* Skills and technology showcase
* Project portfolio section
* Blog section with Read More functionality
* Contact section with social media links

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GitHub Pages


### Option 1: Run Locally

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using a web browser.

### Option 2: Using VS Code

1. Open the project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
