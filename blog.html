Eksperimen 3D Modeling & Animasi Ruangan

Diterbitkan pada: Januari 2026

Proses menghasilkan tugasan animasi 3D berskala kecil...

Proses ini bermula dengan fasa lakaran idea (*storyboarding*) bagi menentukan jalan cerita animasi pembersihan bilik. Seterusnya, proses *low-poly modeling* dilakukan di dalam perisian 3D pilihan untuk memastikan aset bilik kelihatan minimalis dan kemas.

Cabaran utama adalah pada bahagian pencahayaan (*lighting*) bagi menonjolkan mood ruang yang bersih selepas dikemas. Hasil akhir di-eksport ke dalam format .fbx sebelum dilaraskan semula untuk fasa render terakhir.

Membina Re-design UI Portfolio Menggunakan CSS Grid

Diterbitkan pada: Jun 2026

Nota ringkas penataan gaya layout grid digital...

CSS Grid memberikan fleksibiliti yang sangat tinggi untuk menyusun kad responsif tanpa memerlukan kod framework luar yang berat. Corak grid digital halus dibina menggunakan teknik `linear-gradient` pada elemen latar belakang body.

Selain itu, sistem pengurusan warna disentralisasikan menggunakan CSS Variables (`:root`) untuk mempermudahkan peralihan mod gelap (*Dark Mode*) secara automatik merentasi semua fail HTML menggunakan bantuan `localStorage` pelayar.
